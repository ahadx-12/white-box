# TrustAI VRP (White-Box)

This repository hosts the deterministic core for TrustAI VRP (Verifiable Reasoning Protocol).
Week 1 focuses on a deterministic logic kernel built on hyperdimensional computing (HDC),
pack loading, and arbiter evaluation. No API or UI components are included yet.

## Quick start

```bash
python -m pytest -q
ruff check .
mypy packages/core/src/trustai_core
```
